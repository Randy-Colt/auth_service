# Ботоферма

Инфраструктура включает:
- Multi-stage Dockerfile
- Отдельный контейнер для юнит-тестов
- Контейнер приложения
- Контейнер базы данных
- Healthcheck для бекенда и базы


## Запуск проекта

Перейдите в каталог infra:
```
cd infra
```

Выполните сборку и запуск:
```
docker-compose up 
```

Документация Swagger доступна по адресу:
```
http://127.0.0.1:8000/docs
```


## Запуск юнит-тестов

Запускается отдельный контейнер, использующий dev-зависимости.

Выполните:
```
docker-compose --profile test up --build tests
```

Контейнер автоматически завершит работу после завершения тестов.


## Генерация ключей

Ключи для JWT автоматически генерируются при старте контейнера, если их нет. Они сохраняются в volume 'certs'.
